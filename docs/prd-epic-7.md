# Epic 7: Publication & Beta Distribution

**Goal:** Provide the tools necessary to distribute manuscripts to beta readers for feedback and prepare the final publication assets for distribution.

## Scope
This Epic covers the "Distribution" and "Review" phase.
*   **Beta Reader Access:** Secure link sharing with commenting capabilities.
*   **Export Previewer:** WYSIWYG preview of the final PDF/EPUB output.
*   **Metadata Hub:** Centralized form for ISBN, Blurb, Keywords, and Pricing.
*   **Coming Soon Page:** A pre-launch landing page for the book.

## Functional Requirements
*   **Beta Reader Module (PRD 11)**: Invite system + Inline commenting.
*   **Export Engine (PRD 12)**: Stylesheet selection + Real-time preview.
*   **Publication Hub (PRD 13)**: Metadata validation + Distribution readiness check.

## Non-Functional Requirements
*   **Security**: Beta links must be token-based and revocable.
*   **Performance**: Export preview must verify < 5s.
*   **Compatibility**: EPUBs must pass epub-check validation standard.

## User Stories Overview
*   **Story 7.1: Beta Reader Access & Commenting**: Invite beta readers and collect structured feedback.
*   **Story 7.2: WYSIWYG Export Previewer**: Visual check of print/ebook formatting.
*   **Story 7.3: Publication Metadata & Blurb Hub**: Preparing the book's store presence.
*   **Story 7.4: Public Coming Soon Pages**: Pre-launch marketing page.
